# 🎬 Netflix Clone (UI Only)

A simple and modern **Netflix Clone UI** built using **HTML and CSS**.
This project focuses on replicating the look and feel of Netflix’s homepage while practicing frontend fundamentals.

---

## 🚀 Features

* 🔝 **Navbar**

  * Fixed top navigation bar
  * Logo + menu items (Home, TV Shows, Movies, My List)
  * Smooth hover effects

* 🎥 **Banner Section**

  * Full-width hero section
  * Background image with dark gradient overlay
  * Title, description, and action buttons (Play / More Info)

* 🎞️ **Movie Cards Grid**

  * Responsive grid layout
  * Movie posters
  * Hover zoom effect with smooth transitions

* 🎨 **Modern UI**

  * Dark theme (Netflix-style)
  * Clean typography (Google Fonts)
  * Smooth animations

* 📱 **Responsive Design**

  * Works on desktop, tablet, and mobile
  * Grid adjusts based on screen size

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling, layout, animations
* **Flexbox & Grid** – Layout design
